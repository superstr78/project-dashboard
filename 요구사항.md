# 프로젝트 대시보드 요구사항

## 개요
GMDSOFT 제품 관련 프로젝트들의 현황을 한눈에 확인할 수 있는 웹 기반 대시보드

### 회사 정보
- **회사명**: GMDSOFT
- **홈페이지**: https://www.gmdsoft.com/
- **사업 분야**: 모바일 및 디지털 포렌식 솔루션
- **주요 제품**: MD Series, MDRED, Face Search 등

---

## 기능 요구사항

### 1. 패키징 진행 상황 (프로젝트별)
- [ ] 팀즈 패키지별 채널 정보와 연동
- [ ] 상태 표시: `배포 완료` | `진행 중 (1차, 2차 등)`
- [ ] 진행 중인 경우 특이사항 표시 (팀즈 채널 댓글 기준)

### 2. 주요 이슈 상황 (프로젝트별)
- [ ] 팀즈 제품별 이슈 채널 정보와 연동
- [ ] 이슈 목록 및 상태 표시

### 3. 빌드 상태 (프로젝트별)
- [ ] 빌드 성공/실패 여부 표시
- [ ] 실패 시 관련 정보 표시

### 4. SVN 커밋 비율 (프로젝트별)
- [ ] 프로젝트별 커밋 현황 시각화
- [ ] 커밋 비율 통계

### 5. Jira 이슈 현황 (프로젝트별)
- [ ] Jira 연동
- [ ] 최근 생성된 이슈 목록
- [ ] 최근 완료된 이슈 목록

### 6. 설정 페이지
- [ ] 팀즈(Microsoft Teams) 연동 설정
- [ ] Jira 연동 설정
- [ ] Confluence 연동 설정

---

## 페이지 구성

### 메뉴 구조

| 메뉴 | 설명 | 데이터 소스 |
|------|------|-------------|
| 대시보드 | 전체 현황 요약 | 모든 소스 통합 |
| 패키징 현황 | 프로젝트별 패키징 상태 | Teams |
| 이슈 현황 | 제품별 이슈 | Teams |
| 빌드 상태 | 빌드 성공/실패 | 빌드 서버 |
| SVN 커밋 | 커밋 비율 통계 | SVN |
| Jira | 이슈 목록 | Jira API |
| 설정 | 연동 설정 | 로컬 저장 |

### 1. 대시보드 (메인 페이지)
- 전체 현황 요약 카드 (패키징, 빌드, 이슈 통계)
- 프로젝트별 요약 카드
- 최근 활동/알림 목록

### 2. 패키징 현황 페이지
- 프로젝트별 패키징 상태 테이블
- 상태 필터 (전체, 배포완료, 진행중)
- 상세 보기 (특이사항, 최근 댓글)

### 3. 이슈 현황 페이지
- 프로젝트 선택 드롭다운
- 이슈 목록 테이블 (제목, 상태, 담당자, 등록일)

### 4. 빌드 상태 페이지
- 프로젝트별 빌드 상태 테이블
- 성공/실패 표시
- 실패 시 오류 로그 표시

### 5. SVN 커밋 페이지
- 기간 선택 (최근 7일, 30일 등)
- 프로젝트별 커밋 비율 차트
- 최근 커밋 목록

### 6. Jira 이슈 페이지
- 프로젝트/상태 필터
- 최근 생성/완료 탭
- 이슈 목록 테이블

### 7. 설정 페이지
- Teams, Jira, Confluence, SVN, 빌드서버 탭
- 각 서비스별 연동 설정 폼
- 연결 테스트 기능
